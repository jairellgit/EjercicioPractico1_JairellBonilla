Instrucciones rápidas:
1. Importa el proyecto en NetBeans como proyecto Maven (o desde la terminal con 'mvn spring-boot:run').
2. Actualiza 'src/main/resources/application.properties' con tu usuario y contraseña de MySQL.
3. Asegúrate de que la base de datos 'biblioteca' ya existe y contiene las tablas (tuya ya creada).
4. Ejecuta. La app intentará correr en el puerto 78 (server.port=78).
5. Rutas principales:
   - /         -> Página de inicio
   - /libros   -> CRUD de libros
   - /categorias -> CRUD de categorías
   - /quejas/nuevo -> Formulario de quejas/sugerencias
